**MonoX** is a sleek and modern full-stack e-commerce platform built using **ReactJS**, **Node.js**, **Express**, and **MongoDB**.  
It allows users to explore, add to cart, and order products with multiple payment options like **Razorpay**, **Stripe**, and **Cash on Delivery**.  
Currently, **Razorpay** and **Stripe** are integrated but not yet activated.

---

## üöÄ Project Overview

MonoX is divided into three main sections:
1. **Frontend** ‚Äì User interface built with ReactJS, CSS, and Bootstrap.  
2. **Backend** ‚Äì RESTful API using Node.js, Express, and MongoDB for managing users, products, carts, and orders.  
3. **Admin Panel** ‚Äì Admin dashboard for managing products, orders, and users.
